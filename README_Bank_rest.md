# Система управления банковскими картами

## Сборка
    git clone https://github.com/winroot333/bank_rest.git
    cd bank_rest
    mvn -DskipTests=true clean package
## Запуск
Postgres в контейнере:

    docker compose up -d

Запуск приложения:

    java -jar /out/application.jar

Swagger:

    http://localhost:8080/swagger-ui/index.htm
Данные для проверки:
Админ:

    admin
    password
Пользователь с транзакциями и картами:

    user1
    password
    
## Описание:
1. [ ] Через миграцию создаются таблицы и заполняются тестовые данные.
2. [ ] Тесты репозиториев сделаны с Testcontainers
3. [ ] Номер карты хранится в бд  в зашифрованном виде. Masked number также хранится в бд для простоты и производительности.
4. [ ] Мэппинг Dto в сущности и обратно сделан с помощью Mapstruct
5. [ ] Подключен Spring Security с jwt токенами. Для простоты refresh token реализовывать не стал.
6. [ ] Postgres поднимается в контейнере, приложение без контейнера для простоты.
7. [ ] Логгирование в файл и консоль.
8. [ ] У сущности транзакций есть enum со статусом, но используется только COMPLETED. 
9. [ ] openapi.yaml сделал просто, скачал этот файл после генерации из аннотаций swagger ui. Хотел генерировать по нему интерфейсы контроллеров и дтошки, но слишком большой файл получался.
