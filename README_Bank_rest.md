# Система управления банковскими картами

Через миграцию создаются таблицы и заполняются тестовые данные.

Тесты репозиториев сделаны с Testcontainers

Номер карты хранится в бд  в зашифрованном виде. Masked number также хранится в бд для простоты и производительности.

Мэппинг Dto в сущности и обратно сделан с помощью Mapstruct

Подключен Spring Security с jwt токенами. Для простоты refresh token реализовывать не стал.

Postgres поднимается в контейнере, приложение без для простоты.

Логгирование в файл и консоль.

У сущности транзакций есть enum со статусом, но используется только COMPLETED.
